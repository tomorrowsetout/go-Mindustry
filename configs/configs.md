json                     # 可读存储性修改存放各种各样的信息
api.ini                  # API主要配文件也存放API密钥  
config.ini               # mdt-server 配置总控文件 只放全局控制项，不堆放业务配置
core.ini                 # mdt-server 核心配置 存放双核调度等等 内存释放控制
Development mode.ini     # 开发模式配置 主要存放的都是开发者日志存放的开关 比如监听某些事件等等
misc.ini                 # 杂项配置 没多大用默认不修改即可
Personalization.ini      # 个性化显示配置 主要显示的就是终端配置样式等等 默认不修改即可
release.ini              # 释放控制 它会释放包含配置文件等多数文件的重新释放覆盖只要重新启动即可
server.ini               # server 基础信息配置 服务器名称 服务器简介虚拟玩家人数都在这里配置
Sundries.ini             # 日志文件配置 主要存放的是日志输出后的内容 默认不修改即可 主要存放的就是 单个文件多大最多允许多少个文件
sync.ini                 # 快照与网络同步配置 默认不修改即可
player.toml              # 存放玩家配置文件主要管理玩家的配置 比如是否创建相关文件 隐藏真实uuid等等
