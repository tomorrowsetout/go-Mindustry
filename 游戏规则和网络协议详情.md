## 游戏核心概念

### 1.1 游戏模式

Mindustry 是一款塔防+RTS游戏，包含以下主要模式：

- **沙盒模式 (Sandbox)**: 无限资源，无敌人
- **战役模式 (Campaign)**: 任务驱动，资源有限
- **生存模式 (Survival)**: 波次敌人攻击
- **合作模式 (Co-op)**: 多人协作
- **PVP模式 (PvP)**: 玩家对战

### 1.2 团队系统 (Team System)

游戏支持6个基础队伍：

| 团队ID | 团队名称 | 颜色 | 描述 |
|--------|---------|------|------|
| 0 | derelict (废弃) | 灰色 | 自动防御系统 |
| 1 | sharded (碎片) | 蓝色 | 基础中立队伍 |
| 2 | crux (核心) | 红色 | 敌对队伍 |
| 3 | malis (恶意) | 紫色 | 强敌队伍 |
| 4 | green (绿色) | 绿色 | 中立友好 |
| 5 | blue (蓝色) | 靛蓝 | 中立友好 |

**关键特性:**
- 每个团队共享核心资源
- 核心方块生成该团队的单位
- 团队间有敌对/友好关系

### 2.3 玩家死亡机制

#### 死亡检测流程：

```java
// PlayerComp.java 中的实现
public void update(){
    if(!dead()){
        // 玩家还活着，附身单位
        set(unit);
        unit.team(team);
        deathTimer = 0;
    }else if((core = bestCore()) != null){
        // 玩家已死亡，等待重生
        deathTimer += Time.delta;
        if(deathTimer >= deathDelay){  // 60 ticks = 1秒
            core.requestSpawn(self());  // 请求重生
            deathTimer = 0;
        }
    }
}

public boolean dead(){
    return unit == null || !unit.isValid();
}
```

#### 死亡后的处理：

1. **死亡瞬间** (0-1秒):
   - 单位被清除 (`clearUnit()`)
   - 视角保持在死亡位置
   - 可以看到死亡图标

2. **重生倒计时** (1秒后):
   - 自动寻找最近的核心
   - 计时器达到阈值后请求重生
   - 核心生成新单位

3. **重生完成**:
   - 新单位在核心位置生成
   - 玩家附身到新单位
   - 视角切换到新单位

### 2.4 核心选择算法

```java
// PlayerComp.java - bestCore() 方法
public CoreBuild bestCore(){
    var cores = team.cores();
    
    // 优先选择支持当前环境的核心
    // 如果只有一个核心，优先选择它
    // 如果有多个核心，优先选择大核心，其次近核心
    return cores.min(b -> 
        cores.size == 1 || 
        ((CoreBlock)b.block).unitType.supportsEnv(state.rules.env),
        Structs.comps(
            Structs.comparingInt(c -> -c.block.size),  // 大核心优先
            Structs.comparingFloat(c -> c.dst2(x, y))   // 近核心优先
        )
    );
}
```

**选择规则:**
1. 环境支持检查（太空核心不能在地面使用）
2. 核心大小优先（更大的核心）  
3. 距离优先（更近的核心）

```java
// 玩家附身新单位
public void unit(@Nullable Unit unit){
    // 阻止在刚切换后再次切换
    if(isLocal() && unit == justSwitchFrom && justSwitchFrom != null && justSwitchTo != null){
        return;
    }
    
    if(this.unit == unit) return;
    
    // 保存旧单位的最后一个命令
    if(unit != null && unit.controller() instanceof CommandAI ai){
        lastCommand = ai.command;
    }
    
    if(this.unit != null){
        // 释放旧单位
        this.unit.resetController();
        // 恢复上一个命令
        if(lastCommand != null && this.unit.controller() instanceof CommandAI ai){
            ai.command(lastCommand);
        }
    }
    
    this.unit = unit;
    if(unit != null){
        unit.team(team);
        unit.controller(this);  // 设置控制器
        unit.snapInterpolation();  // 瞬间同步位置
    }
    
    Events.fire(new UnitChangeEvent(self(), unit));
}
```

```java
// Turret.java
public class Turret extends ReloadTurret{
    public ObjectMap<Item, BulletType> ammoTypes = new OrderedMap<>();
    public float reload = 30f;           // 装填时间
    public float shootRange = 0f;        // 射击范围
    public float laserDuration = 0f;     // 激光持续时间
    
    // 处理物品弹药
    public void handleItem(Building source, Item item){
        BulletType type = ammoTypes.get(item);
        if(type == null) return;
        
        totalAmmo += type.ammoMultiplier;
        
        // 查找或创建弹药条目
        for(int i = 0; i < ammo.size; i++){
            ItemEntry entry = (ItemEntry)ammo.get(i);
            if(entry.item == item){
                entry.amount += type.ammoMultiplier;
                return;
            }
        }
        ammo.add(new ItemEntry(item, (int)type.ammoMultiplier));
    }
    
    // 热量管理
    public void updateCooling(){
        if(canReload() && coolant != null){
            float capacity = liquid.heatCapacity;
            float amount = liquid.amount * liquid.efficiency(this);
            reloadCounter += amount * edelta() * capacity * coolantMultiplier;
        }
    }
}
```

```java
// PointDefenseTurret.java
public class PointDefenseTurret extends BaseTurret{
    public float retargetTime = 20f;  // 重新定位时间
    
    // 寻找可拦截的子弹
    @Override
    public void updateTargeting(){
        if(timer(timerTarget, retargetTime)){
            target = Groups.bullet.intersect(x - range, y - range, range*2, range*2)
                .min(b -> b.team != team && b.type().hittable, 
                     b -> b.dst2(this));
        }
        
        if(target != null && target.within(this, range) && target.team != team){
            float dest = angleTo(target);
            rotation = Angles.moveToward(rotation, dest, rotateSpeed);
            
            if(Angles.within(rotation, dest, shootCone) && canReload()){
                float realDamage = bulletDamage * state.rules.blockDamage(team);
                
                if(target.damage() > realDamage){
                    target.damage(target.damage() - realDamage);  // 减少子弹伤害
                }else{
                    target.remove();  // 销毁子弹
                }
                
                // 播放拦截效果
            }
        }
    }
}
```

### 6.1 网络架构

```

├── Packet (数据包基类)
│   ├── ConnectPacket (连接数据包)
│   ├── Disconnect (断开连接)
│   ├── StreamBegin (流传输开始)
│   ├── StreamChunk (流数据块)
│   └── Generated Packets (自动生成)
├── Net (网络主类)
│   ├── NetServer (服务器实现)
│   └── NetClient (客户端实现)
├── NetConnection (连接管理)
└── NetworkIO (I/O处理)
```

### 6.2 数据包优先级系统

```java
// Packet.java
public abstract class Packet{
    public static final int priorityLow = 0;    // 低优先级
    public static final int priorityNormal = 1; // 普通优先级
    public static final int priorityHigh = 2;   // 高优先级
    
    public int getPriority(){
        return priorityNormal;
    }
}
```

**优先级说明:**
- **priorityHigh (2)**: 立即处理，用于关键事件（连接、断开）
- **priorityNormal (1)**: 排队处理，用于常规同步
- **priorityLow (0)**: 仅在连接后处理

### 6.3 连接流程

```
客户端连接流程:
1. ConnectPacket (高优先级)
   ├── version (游戏版本)
   ├── versionType (版本类型)
   ├── mods (安装的模组)
   ├── name (玩家名称)
   ├── uuid (UUID)
   ├── usid (用户唯一ID)
   ├── mobile (是否移动端)
   └── color (玩家颜色)

2. WorldStream (流传输)
   ├── StreamBegin (流开始)
   └── multiple StreamChunk (数据块)

3. Sync (同步)
   └── UnitSnapshot (单位快照)

服务器响应:
1. NotifyJoin (加入通知)
2. GameState (游戏状态)
3. EntityData (实体数据)
```

**序列化过程:**
```java
// 写入数据
public void write(Writes buffer){
    buffer.i(version);                    // 4 bytes
    TypeIO.writeString(buffer, type);     // 2 + length bytes
    TypeIO.writeString(buffer, name);     // 2 + length bytes
    TypeIO.writeString(buffer, locale);   // 2 + length bytes
    TypeIO.writeString(buffer, usid);     // 2 + length bytes
    
    byte[] b = Base64Coder.decode(uuid);  // 16 bytes
    buffer.b(b);                          // 16 bytes
    CRC32 crc = new CRC32();
    crc.update(b);
    buffer.l(crc.getValue());             // 8 bytes
    
    buffer.b(mobile ? 1 : 0);             // 1 byte
    buffer.i(color);                      // 4 bytes
    buffer.b((byte)mods.size);            // 1 byte
    
    for(int i = 0; i < mods.size; i++){
        TypeIO.writeString(buffer, mods.get(i));  // variable
    }
}

// 读取数据
public void read(Reads buffer){
    version = buffer.i();
    versionType = TypeIO.readString(buffer);
    name = TypeIO.readString(buffer);
    locale = TypeIO.readString(buffer);
    usid = TypeIO.readString(buffer);
    
    byte[] idbytes = buffer.b(16);
    uuid = new String(Base64Coder.encode(idbytes));
    
    mobile = buffer.b() == 1;
    color = buffer.i();
    
    int totalMods = buffer.b();
    mods = new Seq<>(totalMods);
    for(int i = 0; i < totalMods; i++){
        mods.add(TypeIO.readString(buffer));
    }
}
```

```java
// 玩家死亡和重生的完整流程
public class PlayerComp implements UnitController{
    static final float deathDelay = 60f;  // 1秒倒计时
    float deathTimer;
    
    // 检查玩家是否死亡
    boolean dead(){
        return unit == null || !unit.isValid();
    }
    
    // 更新玩家状态
    @Override
    public void update(){
        if(!dead()){
            // 玩家还活着
            set(unit);
            unit.team(team);
            deathTimer = 0;
            
            // 更新移动
            if(unit.type.canBoost){
                boolean shouldBoost = unit.onSolid() || boosting || 
                                     (unit.isFlying() && !unit.canLand());
                unit.elevation = Mathf.approachDelta(
                    unit.elevation,
                    shouldBoost ? 1f : 0f,
                    shouldBoost ? unit.type.riseSpeed : unit.type.descentSpeed
                );
            }
        }else if((core = bestCore()) != null){
            // 玩家死亡，等待重生
            deathTimer += Time.delta;
            
            if(deathTimer >= deathDelay){
                // 请求重生
                core.requestSpawn(self());
                deathTimer = 0;
            }
        }
    }
    
    // 选择最佳核心
    @Nullable
    CoreBuild bestCore(){
        var cores = team.cores();
        
        // 优先环境支持，其次大小，最后距离
        return cores.min(b -> 
            cores.size == 1 || 
            ((CoreBlock)b.block).unitType.supportsEnv(state.rules.env),
            Structs.comps(
                Structs.comparingInt(c -> -c.block.size),
                Structs.comparingFloat(c -> c.dst2(x, y))
            )
        );
    }
    
    // 处理单位变更
    @Override
    public void unit(@Nullable Unit unit){
        // 防止快速切换
        if(isLocal() && unit == justSwitchFrom && justSwitchFrom != null && justSwitchTo != null){
            return;
        }
        
        // 保存旧单位的命令
        if(unit != null && unit.controller() instanceof CommandAI ai){
            lastCommand = ai.command;
        }
        
        if(this.unit != null){
            // 释放旧单位
            this.unit.resetController();
            
            // 恢复命令
            if(lastCommand != null && this.unit.controller() instanceof CommandAI ai){
                ai.command(lastCommand);
            }
        }
        
        this.unit = unit;
        
        if(unit != null){
            unit.team(team);
            unit.controller(this);
            
            // 瞬间同步位置
            if(unit.isRemote()){
                unit.snapInterpolation();
            }
            
            // 重置选中方块
            if(isLocal()){
                control.input.block = null;
            }
        }
        
        Events.fire(new UnitChangeEvent(self(), unit));
    }
}
```

```java
    public float rotation = 90f;
    public float reloadCounter = 0f;
    public float warmup = 0f;
    public boolean isShooting = false;
    
    // 更新武器
    public void updateReload(){
        // 检查是否可以装填
        boolean canReload = this.canReload();
        boolean hasAmmo = this.hasAmmo();

            if(reloadCounter >= reload){
                // 寻找目标
                Teamc target = findTarget(x, y, range(), bullet.collidesAir, bullet.collidesGround);
                
                if(target != null && target.within(x, y, range())){
                    // 射击
                    float angle = angleTo(target);
                    
                    if(Angles.within(rotation, angle, shootCone)){
                        // 发射子弹
                        shoot(x + Angles.trnsx(rotation, shootX, shootY),
                              y + Angles.trnsy(rotation, shootX, shootY),
                              angle);
                        
                        // 重置
                        reloadCounter = 0;
                    }
                }
            }
        }else if(!canReload || !hasAmmo){
            reloadCounter = 0;
        }
    }
    
    // 射击方法
    public void shoot(float x, float y, float angle){
        // 播放音效
        shootEffect.at(x, y, rotation);
        shootSound.at(x, y, 1f, 1f);
        
        // 创建子弹
        Bullet b = Bull.deploy(x, y, rotation, team, null, 
                               bullet.damage, bullet.len);
        
        // 添加误差
        if(bullet.inaccuracy > 0){
            b.adjust(bullet.inaccuracy * (Math.random() - 0.5f),
                     bullet.inaccuracy * (Math.random() - 0.5f));
        }
        
        // 增加后坐力
        impulse(-Angles.trnsx(rotation, shootRecoil, shootRecoil),
                -Angles.trnsy(rotation, shootRecoil, shootRecoil));
        
        // 播放烟雾
        smokeEffect.at(x + Angles.trnsx(rotation, 6f), 
                       y + Angles.trnsy(rotation, 6f));
    }
}
```

```java
// 单位快照类
public class UnitSnapshot extends Packet{
    public int id;
    public long time;
    public Seq<UnitEntry> units = new Seq<>();
    
    public static class UnitEntry{
        public int id;
        public float x, y;
        public float rotation;
        public float health;
        public float maxHealth;
        public float ammo;
        public Team team;
        public Seq<ItemEntry> items;
        
        public UnitEntry(Unit unit){
            id = unit.id;
            x = unit.x;
            y = unit.y;
            rotation = unit.rotation;
            health = unit.health;
            maxHealth = unit.maxHealth;
            ammo = unit.ammo();
            team = unit.team;
            
            if(unit.items() != null){
                items = new Seq<>();
                for(int i = 0; i < unit.items().currency.size; i++){
                    items.add(new ItemEntry(unit.items().currency.items[i], 
                                           unit.items().currency.amounts[i]));
                }
            }
        }
    }
    
    public static class ItemEntry{
        public Item item;
        public float amount;
        
        public ItemEntry(Item item, float amount){
            this.item = item;
            this.amount = amount;
        }
        
        @Override
        public void write(Writes buffer){
            buffer.s(item.id);
            buffer.f(amount);
        }
        
        @Override
        public void read(Reads buffer){
            item = content.item(buffer.s());
            amount = buffer.f();
        }
    }
    
    @Override
    public void write(Writes buffer){
        buffer.i(id);
        buffer.l(time);
        buffer.s((short)units.size);
        
        for(UnitEntry entry : units){
            buffer.i(entry.id);
            buffer.f(entry.x);
            buffer.f(entry.y);
            buffer.f(entry.rotation);
            buffer.f(entry.health);
            buffer.f(entry.maxHealth);
            buffer.f(entry.ammo);
            buffer.b(entry.team.id);
            
            if(entry.items != null){
                buffer.s((short)entry.items.size);
                for(ItemEntry item : entry.items){
                    item.write(buffer);
                }
            }else{
                buffer.s((short)0);
            }
        }
    }
    
    @Override
    public void read(Reads buffer){
        id = buffer.i();
        time = buffer.l();
        int count = buffer.s();
        units.clear();
        
        for(int i = 0; i < count; i++){
            UnitEntry entry = new UnitEntry();
            entry.id = buffer.i();
            entry.x = buffer.f();
            entry.y = buffer.f();
            entry.rotation = buffer.f();
            entry.health = buffer.f();
            entry.maxHealth = buffer.f();
            entry.ammo = buffer.f();
            entry.team = Team.all[buffer.b()];
            
            int itemCount = buffer.s();
            if(itemCount > 0){
                entry.items = new Seq<>();
                for(int j = 0; j < itemCount; j++){
                    ItemEntry item = new ItemEntry();
                    item.read(buffer);
                    entry.items.add(item);
                }
            }
            
            units.add(entry);
        }
    }
    
    @Override
    public void handleClient(){
        // 客户端处理快照
        for(UnitEntry entry : units){
            Unit unit = Groups.findUnit(entry.id);
            if(unit != null){
                // 线性插值更新
                unit.x = Mathf.lerp(unit.x, entry.x, 0.3f);
                unit.y = Mathf.lerp(unit.y, entry.y, 0.3f);
                unit.rotation = Angles.lerp(unit.rotation, entry.rotation, 0.3f);
                unit.health = entry.health;
                unit.ammo(entry.ammo);
            }
        }
    }
}
```

```java
public class NetServer{
    public void handleInput(NetConnection con, ClientSnapshot packet){
        // 验证数据包
        if(!validateSnapshot(con, packet)) return;
        
        // 获取玩家
        Player player = getPlayerByConnection(con);
        if(player == null || player.unit() == null) return;
        
        Unit unit = player.unit();
        
        // 更新单位状态
        unit.isShooting = packet.shooting;
        
        if(unit.type.canBoost){
            unit.elevation = packet.boosting ? 1f : 0f;
        }
        
        // 处理建造
        for(BuildPlan plan : packet.buildPlans){
            handleBuildRequest(player, plan);
        }
    }
    
    public boolean validateSnapshot(NetConnection con, ClientSnapshot packet){
        // 检查玩家是否存在
        Player player = getPlayerByConnection(con);
        if(player == null) return false;
        
        // 检查是否有单位
        if(player.unit() == null) return true;  // 玩家死亡不算错误
        
        Unit unit = player.unit();
        
        // 检查距离
        if(unit.dst(packet.mouseX, packet.mouseY) > 500){
            return false;  // 可能是作弊
        }
        
        // 检查构建请求
        for(BuildPlan plan : packet.buildPlans){
            if(!player.canBuildAt(plan.x, plan.y)) return false;
            if(!unit.items().has(plan.block.requirements)) return false;
        }
        
        return true;
    }
    
    public void handleBuildRequest(Player player, BuildPlan plan){
        Unit unit = player.unit();
        
        // 检查是否在建造
        if(!unit.canBuild()) return;
        
        // 检查物品
        if(!unit.items().has(plan.block.requirements)) return;
        
        // 检查冷却
        if(unit.buildPlanCooldown > 0) return;
        
        // 消耗物品
        unit.items().remove(plan.block.requirements);
        
        // 创建构建计划
        world.buildPlan(plan.x, plan.y, plan.block, plan.rotation, plan.config);
        
        // 设置冷却
        unit.buildPlanCooldown = plan.block.buildTime / 10f;
    }
}
```

## 总结

### 关键规则总结

1. **玩家死亡**: 1秒后在最近的核心重生
2. **核心选择**: 环境支持 > 核心大小 > 距离
3. **单位控制**: 玩家控制 → CommandAI → AIController
4. **网络同步**: 客户端输入 → 服务器验证 → 广播快照
5. **炮塔射击**: 寻找目标 → 预热 → 检查角度 → 发射

### 网络协议要点

1. **高优先级**: 连接、断开、踢出
2. **普通优先级**: 同步、聊天、构建
3. **低优先级**: 其他数据

### 性能优化

1. **快照频率**: 10-30 Hz
2. **插值**: 0.3-0.5 插值因子
3. **流传输**: 大数据分块传输
4. **组队**: 多个单位批量同步
