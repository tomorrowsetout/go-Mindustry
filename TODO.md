## 快速开始

```bash
# 编译
go build -o mdt-server.exe cmd/mdt-server/main.go

# 运行
./mdt-server.exe

# 指定配置
./mdt-server.exe -config my-config.json

# 查看版本
./mdt-server.exe --version
```
